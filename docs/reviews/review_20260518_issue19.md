## 🟠 Major Issues

### [M-1] `hooks-ci.yml` の `push` トリガーに `.github/workflows/hooks-ci.yml` が含まれていない

**ファイル**: `.github/workflows/hooks-ci.yml:11-15`

```yaml
# 現在のコード — push 側に workflow ファイル自身が含まれない
  push:
    branches: [main]
    paths:
      - "hooks/**"
      - "tasks/hooks.toml"
      - "mise.toml"
```

`pull_request` 側には `".github/workflows/hooks-ci.yml"` が含まれているが、`push` 側にない。
workflow ファイル自体を修正して main にマージしても CI が再トリガーされないため、
workflow の変更が意図通りに動いているかを main push で確認できない。

```yaml
# ✅ 修正案
  push:
    branches: [main]
    paths:
      - "hooks/**"
      - "tasks/hooks.toml"
      - "mise.toml"
      - ".github/workflows/hooks-ci.yml"
```
