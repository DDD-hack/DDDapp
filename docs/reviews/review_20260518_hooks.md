## 🟠 Major Issues

### [M-1] `fmt.Sscanf` のエラーを無視している

**ファイル**: `hooks/main.go:33`
**問題**: 環境変数 `DDD_THRESHOLD_BPM` に `"abc"` など数値でない値が入ったとき、`Sscanf` は失敗するが `threshold` はデフォルト値のまま変わらず、エラーも出ない。意図しない挙動が気づきにくい。

```go
// ❌ 現在のコード
if v := os.Getenv("DDD_THRESHOLD_BPM"); v != "" {
    fmt.Sscanf(v, "%d", &threshold)
}
```

```go
// ✅ 修正案（strconv.Atoi でエラーを検知）
if v := os.Getenv("DDD_THRESHOLD_BPM"); v != "" {
    if n, err := strconv.Atoi(v); err == nil {
        threshold = n
    } else {
        warn(fmt.Sprintf("⚠ Invalid DDD_THRESHOLD_BPM=%q, using default %d", v, defaultThreshold))
    }
}
```
