
### [S-2] `progressBar` のカラーを呼び出し側で指定できるようにする

`printBandStats` では件数をシアンで表示しているのに、バーはグリーンで描画されており視覚的に統一感がない。

```go
// 💡 色を引数にする
func progressBar(pct, width int, color string) string {
    filled := pct * width / 100
    return color + strings.Repeat("█", filled) + colorReset + strings.Repeat("░", width-filled)
}

// 呼び出し側
bar := progressBar(pct, 10, colorCyan)   // 分布表示
bar := progressBar(pct, 10, colorGreen)  // 成功率表示（将来用途）
```
